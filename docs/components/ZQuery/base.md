### `z-query`的属性值

| 参数       | 说明                   | 类型   | 默认值 |
| ---------- | ---------------------- | ------ | ------ |
| items      | 查询框里搜索框的数据源 | array  | -      |
| formData   | 查询框的数据对象       | Object | -      |
| labelWidth | 查询框标题宽度         | number | 80     |

### `items`的属性值

| 参数  | 说明                               | 类型   | 默认值 |
| ----- | ---------------------------------- | ------ | ------ |
| prop  | 查询字段名，对应 slot 组件插槽名称 | string | -      |
| label | 名称                               | string | -      |

### 事件

| 事件名称 | 说明 | 回调参数 |
| -------- | ---- | -------- |
| search   | 查询 |          |
| reset    | 重置 |          |
