### `z-layout`的属性值

| 参数       | 说明             | 类型    | 默认值 |
| ---------- | ---------------- | ------- | ------ |
| left       | 左侧宽度         | number  | 200    |
| right      | 右侧宽度         | number  | 200    |
| top        | 上侧高度         | number  | 60     |
| bottom     | 下侧高度         | number  | 60     |
| leftMove   | 允许左侧变动大小 | Boolean | true   |
| rightMove  | 允许右侧变动大小 | Boolean | true   |
| topMove    | 允许上侧变动大小 | Boolean | true   |
| bottomMove | 允许下侧变动大小 | Boolean | true   |
